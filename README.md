## Presentation
Cub3D is a project by 42 School, whose goal is to create a simplified 3D engine based on ray casting technology, allowing users to explore a maze from a first-person perspective.
It focuses on the use of MiniLibX, the practical application of mathematics to generate a 3D perspective, as well as event management and user interactions.

## Usage
```
make bonus
# Compiles the executable 'cub3D_bonus'

./cub3D_bonus maps/bonus/
# Launches the game with the maps from the folder provided as a parameter
```
